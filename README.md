# Menu Program Python Sederhana

## Fitur

Program ini memiliki 5 menu yang dapat diakses melalui menu:

| No. | Fitur | Deskripsi |
|---|---|---|
| 1 | **Deret Fibonacci** | Menampilkan `n` suku pertama dari deret Fibonacci. |
| 2 | **Cek Ganjil/Genap** | Memeriksa apakah sebuah bilangan yang dimasukkan adalah ganjil atau genap. |
| 3 | **Cek Bilangan Prima** | Menentukan apakah sebuah bilangan termasuk bilangan prima. |
| 4 | **Hitung Huruf Vokal** | Menghitung jumlah huruf vokal (a, i, u, e, o) dalam sebuah kalimat. |
| 5 | **Hitung Pangkat Manual** | Menghitung hasil pemangkatan (pangkat non-negatif) secara manual menggunakan perulangan `while`. |

## Persyaratan

-   Python 3.10+
